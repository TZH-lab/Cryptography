实验报告

一、实验目的

理解流密码密钥复用的安全漏洞，掌握通过已知明文推断密钥流、解密目标密文的方法。

二、实验原理

流密码加密：c_i = m_i \oplus k_i
同一密钥加密多条明文时：c_1 \oplus c_2 = m_1 \oplus m_2
已知明文与对应密文异或可直接恢复密钥流：k_i = c_i \oplus m_i。

三、实验方法

1. 数据预处理：使用 binascii 库将所有十六进制格式的密文转换为字节数组，方便逐字节进行异或运算。

2. 长度对齐：截取与目标密文长度一致（83字节）的已知明文片段，保证逐位运算匹配。

3. 密钥流推导：根据流密码解密公式，将目标密文与已知明文逐字节进行异或计算，得到对应位置的密钥流。

4. 验证与解密：使用推导出的密钥流对其他密文片段进行解密，验证密钥流有效性；最终解密目标密文得到完整明文，并保存结果。

四、密钥流推断过程
根据流密码加密原理 c=m\oplus k，变形得密钥流 k=c\oplus m。
将目标密文与已知正确明文按字节逐位异或，直接计算出对应位置的密钥流，再用该密钥流验证并解密目标密文。

五、、实验结果

• 成功恢复83字节密钥流。

• 最终明文：
The secret message is: Never use stream cipher key more than once.

六、结论

流密码密钥严禁复用，否则可通过密文与明文异或快速破解，验证了密钥复用的严重安全风险。
